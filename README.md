# SignUp/Login App #

This is a demo app based on a generic process of sign-up and login users

## Concept ##
- Sign Up process
- Login process
- Forget password option


## Version 1 specs ##

### Technical Stack ###
* Data storage and auth agent: Firebase
* FrontEnd: Kotlin for Android

### Available Functions ###
* SignUp
* Login
* Reset password
